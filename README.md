# Spam Email Classifier

This project classifies emails as Spam or Not Spam using Machine Learning and NLP.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- TF-IDF

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python spam_classifier.py

## Output
- Accuracy of model
- Prediction for sample email
